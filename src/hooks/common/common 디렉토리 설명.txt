📁 src/hooks/common/
목적: 앱의 여러 도메인에서 공통적으로 사용될 수 있는 범용 커스텀 훅들을 모아놓은 폴더입니다. 특정 기능에 종속되지 않는 재사용 가능한 로직을 제공합니다.

포함된 훅(Hooks):

useModal.ts: 모달(팝업) 창의 열림/닫힘 상태를 손쉽게 제어하는 훅입니다. openModal, closeModal, toggleModal 함수를 제공하며, 모달에 특정 데이터를 전달하는 기능도 포함합니다.

useCalendar.ts: 캘린더 기능에 필요한 복잡한 상태와 로직을 관리하는 훅입니다. 이벤트 추가/수정/삭제, 날짜 선택, 월 이동 등의 기능을 제공하여 캘린더 UI 컴포넌트를 제어합니다.

useDebounce.ts: 사용자의 잦은 입력이 있을 때, 마지막 입력 후 일정 시간(delay)이 지나야 값을 업데이트하는 디바운싱(Debouncing) 기술을 구현한 훅입니다. 검색창 자동완성 등에서 불필요한 API 호출을 줄여 성능을 최적화하는 데 사용됩니다.

useLocalStorage.ts: 브라우저의 localStorage와 React의 useState를 동기화하는 훅입니다. 이를 통해 특정 상태를 브라우저에 저장하여 페이지를 새로고침해도 값이 유지되도록 할 수 있습니다.

특징:

높은 재사용성: 훅의 로직이 특정 비즈니스에 묶여있지 않아, 모달, 캘린더, 검색창 등 유사한 기능이 필요한 어느 곳에서든 재사용할 수 있습니다.

로직 추상화: 컴포넌트로부터 복잡한 로직을 분리(추상화)하여 UI 컴포넌트는 오직 화면을 그리는 역할에만 집중할 수 있게 합니다.

개발 생산성 향상: 자주 사용되는 기능들을 훅으로 만들어 둠으로써 반복적인 코드 작성을 줄이고 개발 속도를 높여줍니다.

관심사의 분리(Separation of Concerns): UI 로직과 상태 관리 로직을 명확하게 분리하여 코드의 가독성과 유지보수성을 향상시킵니다.
