📁 src/components/layout/

목적: 페이지 레이아웃과 전체적인 구조를 담당하는 컴포넌트들
포함된 컴포넌트들:

Header.tsx - 애플리케이션 헤더 (로고, 사용자 메뉴, 사이드 메뉴 포함)

UserMenu: 사용자 프로필 드롭다운 메뉴
SideMenu: 좌측 사이드바 메뉴 (챗봇 등)


MainLayout.tsx - 메인 페이지 레이아웃 래퍼

PageLayout: 일반 페이지용 레이아웃 (마이페이지 등)


index.ts - 레이아웃 컴포넌트들의 export 진입점

특징:

반응형 디자인 지원
사용자 인증 상태에 따른 조건부 렌더링
메뉴 상태 관리 (열림/닫힘)
일관된 헤더 및 네비게이션 제공
